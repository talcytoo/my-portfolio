# talcytoo.github.io

Personal portfolio website — built with Next.js, React, Tailwind CSS, and deployed as a static site to GitHub Pages.

**Live site:** https://talcytoo.github.io

## Project Structure

```
portfolio/
├── app/
│   ├── globals.css       # Global styles + scrollbar-hide utility
│   ├── layout.tsx        # HTML layout, metadata (tab title)
│   └── page.tsx          # Main page — all project data + components
├── public/               # Static assets (images, GIFs, profile photo)
├── out/                  # Static export output (generated by build)
├── next.config.ts        # Static export config (output: "export")
├── package.json          # Dependencies + build script
├── postcss.config.mjs    # Tailwind PostCSS plugin
└── tsconfig.json         # TypeScript config
```

## Key File

**`app/page.tsx`** contains everything:
- `hero` object — name, title, bio, email, social links, profile image path
- `projects` array — each project has: id, title, images[], tech[], overview[], contributions[], results[], and optional github/externalUrl
- `ImageCarousel` component — horizontal snap-scroll with arrow buttons (appears for projects with 2+ images)
- `RepoButton` component — renders GitHub "View Source" or external link button
- Main layout — left sticky sidebar with profile + table of contents, right scrollable project sections

## Local Development

```bash
npm install
npm run dev          # http://localhost:3000
```

## Deploy to GitHub Pages

The build script automatically creates `.nojekyll` (required so GitHub Pages serves the `_next/` directory).

```bash
npm run build        # builds to out/ and adds .nojekyll

cd out
git init
git checkout -b main
git add -A
git commit -m "Deploy"
git remote add origin https://github.com/talcytoo/talcytoo.github.io.git
git push origin main --force
```

> **Note:** `npm run build` wipes `out/` entirely (including `.git`), so you must re-init git in `out/` after every build.

## Common Changes

| Task | Where to edit |
|---|---|
| Update bio / social links | `hero` object in `app/page.tsx` |
| Add/remove/reorder projects | `projects` array in `app/page.tsx` |
| Change project images | Update `images` array + add files to `public/` |
| Edit tab title | `metadata.title` in `app/layout.tsx` |
| Re-enable contributions/results | Uncomment the section in the JSX (search for "commented out for now") |
| Change global styles | `app/globals.css` |
